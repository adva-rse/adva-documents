Lettre — Michel à Gérard : réalignement stratégique avant Chantier 0
Date : 3 mai 2026
De : Michel, CTO
À : Gérard, Actionnaire Stratégique
Objet : Demande d'arbitrage sur les ambitions stratégiques d'ADVA
1. Constat factuel
L'échange avec Josselin a révélé une tension produit non résolue. ADVA a été construit comme un livrable narratif valorisant (héritage Produit A — "structurer la démarche RSE"), tout en étant vendu comme un livrable transactionnel (pricing one-shot 990€, argumentaire donneur d'ordres — Produit B). Le pivot du 3 mai vers le froid factuel résout cette tension par le bas.
L'expérience vécue par l'employeur de Josselin (PME aérospatiale sous pression Safran) éclaire le marché réel : embauche d'un stagiaire 5 mois, dossier EcoVadis constitué, note basse, entrée dans un cycle annuel d'amélioration continue piloté par le QHSE. Ce cycle n'a pas d'outil dédié sur le marché.
2. Trois ambitions stratégiques distinctes
Ambition A — Structuration de démarche RSE (consultant low-cost). Écartée : ne correspond ni au pricing actuel ni à la crédibilité fondateur.
Ambition B — Génération de matière première VSME structurée. Outil transactionnel froid factuel, pricing 990/1290€ HT. C'est ce qui est implicitement acté dans le pivot.
Ambition C — Colonne vertébrale RSE de la PME industrielle. Génération + plan d'actions par service + suivi 12 mois + mise à jour automatique. Abonnement 200-400€/mois. Application web, base de données relationnelle, multi-utilisateurs. Saut technique massif depuis Make.com. Définition d'un actif productif au sens DeMarco.
3. Proposition soumise à arbitrage
Livrer Ambition B comme MVP commercial (Chantiers 0 à 7 inchangés) tout en actant Ambition C comme intention stratégique long terme. Imposer dès aujourd'hui deux disciplines architecturales sans coût additionnel.
Discipline 1 : Response ID Tally = identifiant client permanent. Lignes Google Sheets jamais purgées. Base de données client pérenne et interrogeable.
Discipline 2 : Le rapport Word n'est qu'une vue figée d'un état. La donnée vit dans Google Sheets. Discipline mentale : nous ne modifions pas le rapport, nous modifions la vue Word.
4. Conséquence sur les 10 premiers clients
Les 10 premiers clients deviennent une étude de marché formalisée. Questionnaire post-livraison standardisé portant sur l'usage interne, l'identité du pilote EcoVadis, l'organisation de la boucle d'amélioration.
5. Risques nommés honnêtement
Saut technique Ambition B → Ambition C massif. Stack à reconstruire le moment venu.
Pari non vérifié sur le QHSE comme utilisateur cible. À valider par les 10 premiers clients.
Tentation de quitter trop tôt l'emploi salarié. Le poste actuel est un observatoire R&D gratuit sur la cible exacte.
6. Demande d'arbitrage
Validation Ambition B = MVP exclusif, Ambition C = vision long terme actée mais non engagée techniquement avant 10 clients ?
Validation des deux disciplines architecturales dès le Chantier 0 ?
Validation de l'étude de marché formalisée sur les 10 premiers clients ?
